Deadlock Detection & Recovery Simulator

The Deadlock Detection & Recovery Simulator is a web-based system-level application designed to demonstrate how deadlocks occur in operating systems and how they can be detected and resolved.

This project focuses on combining core operating system concepts with an interactive web interface, making complex theoretical ideas easier to understand through visualization.

Key Features

  • Deadlock detection using graph-based techniques
  • Dynamic visualization of system state
