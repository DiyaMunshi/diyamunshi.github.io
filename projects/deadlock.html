Deadlock Detection & Recovery Simulator

An interactive web-based simulator designed to visualize and explain deadlocks in operating systems through dynamic graphs and real-time system state updates.

Project Overview

Deadlocks are a fundamental yet abstract concept in operating systems. This project bridges theory and practice by allowing users to simulate process‚Äìresource interactions, detect deadlocks, and observe recovery mechanisms visually.

The simulator models system behavior using a Wait-For Graph (WFG) and applies graph-based logic to identify circular wait conditions.

Key Features

  • Simulation of process and resource allocation
  • Deadlock detection using DFS-based cycle detection
  • Dynamic visualization of system state
  • User-driven recovery by terminating processes
  • Beginner-friendly and educational interface
