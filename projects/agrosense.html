An AI-powered agriculture assistant designed to support better crop management through plant disease detection and intelligent irrigation recommendations.

Project Overview

AgroSense combines machine learning and web technologies to provide data-driven insights that help improve agricultural productivity and sustainability.

The system focuses on early disease identification and optimized water usage, enabling informed decision-making for crop health management.

Core Modules

  • Plant Disease Detection: Image-based disease classification using deep learning and transfer learning.
  • Smart Irrigation Support: Predictive analysis to assist optimal watering decisions.
  • Web Application Interface: Simple, accessible UI for interacting with the system.

How the System Works

  • User uploads a plant leaf image through the web interface
  • Deep learning model analyzes the image for disease detection
  • System displays disease status along with causes and treatment suggestions
  • Irrigation module provides water-related recommendations

Technology Stack

Python Flask TensorFlow Keras MobileNetV2 HTML CSS JavaScript

Use Cases

  • Farmers monitoring crop health and diseases
  • Students learning applied machine learning in agriculture
  • Academic demonstrations of AI-driven farming solutions
